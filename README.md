LEARNIT is a web-based Student Learning Platform designed to support
Ordinary Level (O/L) and Advanced Level (A/L) students studying ICT and
Computer Science. The platform delivers structured learning materials,
multimedia content, assessments, and progress tracking through a
subscription-based model.

## Features

- User registration and authentication
- Subscription-based access control (Free, Basic, Pro)
- Structured ICT and Computer Science courses
- Multimedia learning content (videos, images, PDFs)
- Online quizzes and assessments
- Student progress tracking and analytics
- Discussion and support features
- Admin and instructor content management

## Technology Stack

### Frontend
- Next.js
- TypeScript
- HTML5, CSS3

### Backend
- Node.js
- RESTful APIs

### Database
- PostgreSQL

### Media Storage
- Cloudinary (images, videos, documents)

## User Roles

- **Student**: Access learning materials, complete quizzes, track progress
- **Instructor**: Upload and manage courses and learning materials
- **Administrator**: Manage users, subscriptions, and system settings

## Subscription Plans

- **Free Plan**: Limited access to introductory materials
- **Basic Plan**: Access to standard notes and limited assessments
- **Pro Plan**: Full access to all courses, assessments, certificates, and analytics
